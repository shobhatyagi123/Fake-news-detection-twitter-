 ============================================= FAKE TWEET ============================================
 
                        ##### ========= Faketweet Version 1.0 ========= #####
 
 FakeTweet is a google chrome extension for detection of misinformation on Twitter.
 
 Run the Flask Server python file in Anaconda Prompt.
 Open Chrome --> Customize menu --> More Tools --> Extension
 Search for "FakeTweet" extension and enable it.
 Go to twitter website.
 Click on any tweet you find suspicious.
 Right click and select option "FakeTweet : CHECK credibility"
 You will be notified about the credibility of the tweet.
 
 
 ======================================= SYSTEM & SOFTWARE REQUIREMENTS ==============================
 
 
 - 32-bit / 64-bit PC with minimum 4GB of RAM.
 - Any Operating System such as Windows 7/8/10, Ubuntu.
 - Internet Connection.
 - Google Chrome Browser.
 - Anaconda Distribution with latest Python 3.x version.
 - Additional packages which may or may not be incuded under Anaconda need to be installed using PIP :
    -- Tweepy           : pip install tweepy
    -- Emoji            : pip install emoji
    -- RAKE             : pip install rake-nltk
    -- NLTK             : import nltk --> nltk.download()  # in python prompt ; download all packages
    -- TextBlob         : pip install textblob 
                        : python -m textblob.download_corpora
